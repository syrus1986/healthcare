# Healthcare Data Analysis

This repository contains hospital data collected across multiple cities and departments. The dataset includes information about hospital names, locations, departments, doctor and patient counts, admission/discharge dates, and medical expenses.

---

##  Dataset Overview
- **Hospitals Covered:** Wellness Clinic, Fortis Care, Heritage Hospital, City Hospital, Global Medicare, Apollo Health, Healing Touch, Metro Hospital, Sunrise Medical, Green Valley Hospital  
- **Departments Included:** Orthopedics, ENT, Pediatrics, Urology, Gynecology, Oncology, General Medicine, Dermatology, Neurology, Cardiology  
- **Key Metrics:**
  - Doctors Count per hospital/department
  - Patients Count per hospital/department
  - Admission & Discharge dates
  - Medical Expenses (per case)

---

##  Purpose
The dataset is designed for:
- Analyzing hospital performance across locations and departments  
- Studying patient inflow and doctor availability  
- Tracking medical expenses and resource utilization  
- Supporting healthcare management decisions with data insights
